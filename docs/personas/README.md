## üöÄ Using Agents

### Via Cursor Command

Use the `/neuraforge` command to deploy any agent:

```
/neuraforge [AGENT_NAME]
```

**Examples:**
```
/neuraforge TEST
/neuraforge MCP
/neuraforge FEATURE
/neuraforge ARCHITECT
/neuraforge CODER
```

### Via Direct Prompt

Load agent persona document directly:

```
@docs/personas/[AGENT]_PERSONA.md
```

Then reference the corresponding prompt file in project root.
