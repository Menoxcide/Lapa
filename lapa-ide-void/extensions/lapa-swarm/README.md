# LAPA Swarm Extension

This extension integrates the LAPA (Local AI Pair Programmer Agent) core functionality into VoidChassis as a swarm-based extension.

## Features

- Swarm-based AI pair programming
- Context-aware code suggestions
- Multi-modal interaction capabilities
- Integrated testing and validation
- Real-time collaboration features

## Requirements

- VoidChassis v1.0 or higher
- Node.js 18.x or higher

## Installation

1. Install the extension from the VoidChassis Marketplace
2. Restart VoidChassis to activate the extension

## Usage

- Use the command palette to start the LAPA swarm with "LAPA: Start Swarm"
- Configure settings through the extension settings panel

## Development

This extension was extracted from the LAPA core codebase with 100% path preservation.

To build and test locally:

```bash
npm install
npm run compile
npm run package
```
