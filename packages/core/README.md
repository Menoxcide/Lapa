# @lapa/core

LAPA Core - Local AI Pair Programmer Agent

This package contains the core LAPA functionality that can be used across different platforms.

## Installation

```bash
pnpm install @lapa/core
```

## Usage

```typescript
import * as lapa from '@lapa/core';
```

## Build

```bash
pnpm build
```

## Test

```bash
pnpm test
```
